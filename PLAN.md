# Redesign all animal silhouettes with realistic shapes

**What's changing**

All 23 animal silhouettes will be completely redrawn using detailed, hand-crafted SVG paths instead of the current basic shapes (circles, rectangles, ellipses). Each animal will have a recognizable, realistic outline — you'll be able to tell a lion from a hyena, a shark from a dolphin, etc.

**Animals being redesigned (23 total)**

- **Savanna** (6): Lion, Elephant, Cheetah, Zebra, Giraffe, Hyena
- **Ocean** (6): Orca, Shark, Dolphin, Narwhal, Mantaray, Octopus
- **Jungle** (6): Gorilla, Jaguar, Anaconda, Toucan, Capybara, Poison Frog
- **Arctic** (5): Polar Bear, Arctic Fox, Walrus, Snowy Owl, Wolverine

**Design approach**

- Each animal will be a single filled silhouette path with natural proportions and curves
- Distinctive features preserved: lion's mane, elephant's trunk, giraffe's long neck, shark's dorsal fin, toucan's beak, owl's round face, etc.
- Still colored by biome accent color (same as before)
- Sized within the same 48×48 viewBox so nothing else in the app needs to change
- Side-profile view for most animals for maximum recognizability
